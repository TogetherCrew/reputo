# Fix

## Goal

Take the issue described in the chat (errors, failing tests, unexpected behavior), **fully understand the root cause**, implement the **best minimal-risk fix**, and **verify** the fix with the appropriate build/test/lint steps for this repo.

### Steps

1. Understand the issue (no guessing).
2. Diagnose to the root cause
3. Implement the fix
4. Verify (iterate until green)
5. Final report (required)

### Output

- What was broken (1–2 sentences)
- What changed (high-level)
- One paragraph, concrete and specific
- Bullet list of key code changes (files + intent)
- Commands run + results (paste the exact commands)
- Any edge cases, follow-ups, or things to watch in CI/production
